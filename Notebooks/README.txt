# 7006SCN Machine Learning and Big Data Assignment

## Project Title

Big Data Analytics and Machine Learning Classification using Nifty Options Data

## Student Information

* **Module:** 7006SCN – Machine Learning and Big Data
* **Student ID:** 16839575

## Project Overview

This project presents a complete big data analytics pipeline developed using Apache Spark and PySpark for financial market prediction. The Nifty Options dataset obtained from the Hugging Face repository has been processed to perform binary classification for option price movement. The workflow includes data preprocessing, feature engineering, machine learning model development, hyperparameter tuning, Spark performance analysis, model evaluation, explainability, and interactive Tableau dashboards.

## Dataset

* Dataset: Nifty Options Data
* Source: Hugging Face Repository
* Records Used: 10,000,000
* Features: 15
* Storage Format: Parquet
